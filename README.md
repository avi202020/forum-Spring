#Forum application# 
**Using technologies:** 
- Spring Boot, 
- Hibernate, 
- Spring Data JPA,
- Java 8,
- Angular 5, 
- JUnit
- Mockito 
- PostgreSQL, 
- Gradle



**To run backend:**

in main project directory: `bootRun` - project will run on port 8080

**To run frontend:**

`cd src/main/frontend`

`npm start` (using proxy settings)
